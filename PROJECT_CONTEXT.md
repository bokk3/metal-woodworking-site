# Metal/Woodworking Business Landing Page

## Project Mission
Build a visually stunning, modern landing page that showcases metal and woodworking craftsmanship with an immersive, contemporary design that makes visitors say "wow."

## Tech Stack
- Next.js 16 (App Router)
- React 19
- TypeScript (strict mode)
- Tailwind CSS + shadcn/ui
- Framer Motion (animations)
- SQLite (portfolio/testimonials)
- Docker deployment ready

## Design Philosophy
- **Dark Mode First**: Industrial aesthetic with metallic accents
- **Color Palette**:
  - Background: #0a0a0a, #1a1a1a
  - Primary: Bronze/copper (#cd7f32, #b87333)
  - Secondary: Steel gray (#71797E, #52595D)
  - Accent: Warm amber (#FFA500)
- **Typography**: Bold, industrial fonts
- **Animations**: Purposeful micro-interactions, parallax, smooth scrolls
- **Imagery**: High-quality, optimized photos of metalwork and woodworking

## Key Sections
1. Hero - Full-screen with dynamic visuals
2. Portfolio Gallery - Filterable, animated grid
3. Services - Interactive cards
4. Process Timeline - Visual storytelling
5. Testimonials - Carousel
6. Contact Form - Validated
7. About - Craftsmanship story

## Performance Targets
- Lighthouse: 90+ all metrics
- Mobile-first responsive
- 60fps animations
- Optimized images (next/image)
